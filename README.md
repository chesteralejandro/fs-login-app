# Login Application

A small full-stack authentication application built with Node.js and Express, demonstrating the MVC (Model–View–Controller) pattern, routing, and server-side validation.

## 📝 About This Project

This project is a small full-stack application built with Node.js and Express to practice the MVC (Model–View–Controller) architecture, routing, and server-side validation. It implements a simple authentication flow with user registration and login, culminating in a protected welcome page for authenticated users.

Rather than focusing on production-level complexity, this project emphasizes clean structure, separation of concerns, and the integration of backend logic with server-rendered views — making it a solid learning exercise and a portfolio-ready demonstration of fundamental full-stack concepts.
